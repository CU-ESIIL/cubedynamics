# Vegetation Analysis

Vegetation workflows in CubeDynamics use the same streaming grammar as climate and fire workflows.

Typical use cases include:

- NDVI anomalies
- z-score cubes
- vegetation responses to climate or disturbance

Recommended starting points:

- [Sentinel-2 NDVI z-score cube](../recipes/s2_ndvi_zcube.md)
- [Sentinel-2 tail dependence](../recipes/s2_tail_dep_center.md)
