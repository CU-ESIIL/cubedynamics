# Climate Analysis

Climate workflows in CubeDynamics focus on computation over time-aware environmental grids.

Typical tasks include:

- anomaly computation
- aggregation
- detrending
- rolling statistics
- spatial-temporal comparisons

Recommended starting points:

- [gridMET variance cube](../recipes/gridmet_variance_cube.md)
- [PRISM variance cube](../recipes/prism_variance_cube.md)
