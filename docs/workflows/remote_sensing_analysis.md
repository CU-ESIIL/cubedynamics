# Remote Sensing Analysis

Remote sensing workflows in CubeDynamics focus on computation over Earth observation data streams, not archive management as an end in itself.

Typical use cases include:

- cube assembly for analysis
- anomaly and trend workflows
- event-aware analysis

Recommended starting points:

- [Sentinel-2 correlation center](../recipes/s2_corr_center.md)
- [Landsat 8 (MPC)](../datasets/landsat8_mpc.md)
